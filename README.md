Sales Dataset Cleaning Project

**Overview**

This project demonstrates data cleaning and preparation techniques using Microsoft Excel.The goal was to improve data quality and prepare dataset for analysis.

**Dataset**

The dataset contain retail sales transactions records,including order dates,coupon codes,quantities and revenue related informations.

**Task performed**

Identified 309 missing values in the couponcode column.

Replaced missing couponcodes with No-coupon.

Formatted the date column into a consistent date/month/year format.

Sorted dates from the oldest to the newest records.

Reviewed  and verified calculations to ensure data accuracy.

**Outcome**

The dataset is clean,organized and useful for further analysis and visualization.
